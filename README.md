# BITS Workshop 1: Agentic AI Coding

A hands-on workshop exploring agentic AI coding workflows using OpenCode and modern AI-assisted development tools.

## Getting Started

See [instructions.md](instructions.md) for setup guide, prerequisites, and the project prompt.

## Workshop Overview

- **Goal:** Build a secure, minimalist desktop password manager using Tauri
- **Tools:** OpenCode, AI-assisted workflows, agentic skills
- **Approach:** Learn how AI agents can scaffold, iterate, and refine real-world applications

## Skills & Plugins

This repo includes pre-configured agentic skills for planning, testing, code review, UI/UX design, and more. Explore the `.agents/` directory to see what's available.
